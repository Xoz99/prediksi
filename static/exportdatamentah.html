Format File CSV:

File CSV harus berisi kolom-kolom berikut:

  • Tahun - Tahun pendaftaran/registrasi
  • Lembaga - Nama fakultas atau lembaga
  • Realisasi Pendaftar - Jumlah pendaftar pada tahun tersebut
  • Realisasi Registrasi (NIM) - Jumlah mahasiswa yang melakukan registrasi

Bantuan

Tentang Data Mentah dan Matang

Data Mentah: Data asli dengan format dasar yang belum diproses.

Data Matang: Data yang telah diproses, dibersihkan, dan ditransformasi menjadi format yang lebih mudah dianalisis.

Proses Transformasi

  • Pembersihan data dari nilai yang tidak valid
  • Penghitungan rasio konversi (Registrasi/Pendaftar)
  • Pengelompokan berdasarkan lembaga dan tahun
  • Perhitungan tren dan pertumbuhan
  • Prediksi untuk tahun-tahun berikutnya
