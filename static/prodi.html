Upload Data

Upload file CSV atau Excel yang berisi data nilai siswa untuk melihat distribusi program studi.

Petunjuk Penggunaan:

  • File harus dalam format CSV atau Excel (.xlsx)
  • File harus memiliki kolom Program yang berisi nama program studi
  • Tambahkan kolom tahun untuk melihat visualisasi berdasarkan tahun
  • Jika mengalami masalah, gunakan tombol "Tes Upload" untuk memeriksa file
  • Atau gunakan tombol "Lihat Data Contoh" untuk mengunduh template data
  • Gunakan tombol "Prediksi PMB Compare" untuk membandingkan prediksi penerimaan mahasiswa baru

Hasil Analisis

Hasil analisis distribusi program studi akan muncul di sini setelah Anda mengupload data.

Bantuan & Dukungan

Jika Anda memerlukan bantuan dalam menggunakan sistem ini, silakan lihat panduan berikut:

Format File yang Didukung

Sistem ini mendukung format file berikut:

  • CSV (Comma Separated Values)
  • Excel (.xlsx, .xls)

Pastikan file Anda memiliki setidaknya kolom "Program" yang berisi nama program studi.

Cara Menganalisis Data
  1. Upload file CSV atau Excel Anda menggunakan form upload
  2. Klik tombol "Upload dan Analisis"
  3. Sistem akan memproses data dan menampilkan hasil analisis
  4. Anda dapat melihat distribusi program studi dan visualisasi data
Masalah Umum dan Solusi

File tidak dapat diupload: Pastikan format file sesuai (.csv, .xlsx, .xls)

Data tidak muncul: Periksa apakah file memiliki kolom "Program"

Visualisasi tidak muncul: Pastikan kolom "tahun" ada dalam file data Anda
