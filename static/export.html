Informasi Format Data

Tool ini mengubah data mentah (format lebar) menjadi format yang diproses (format panjang) untuk digunakan dengan model AI prediktif Telkom University.

Format Input:
  • File CSV atau Excel dengan kolom Program/PRODI
  • Kolom tahun (seperti "Reg NIM '20/21", dll)
  • Data harus tersusun dengan baik tanpa sel-sel kosong
  • Header harus ada pada baris pertama
Format Output:
  • File CSV dengan kolom: Program, Year, Reg_NIM
  • Termasuk data historis dan prediksi untuk tahun-tahun mendatang
  • Format siap untuk diintegrasikan dengan model machine learning

Tips:

Pastikan data Anda bersih dan lengkap untuk hasil prediksi yang optimal. Jumlah tahun prediksi yang disarankan adalah 3-5 tahun untuk akurasi terbaik.
