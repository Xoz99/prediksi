Prediksi Pendaftaran dan Registrasi

Sistem ini memungkinkan Anda untuk memprediksi jumlah pendaftar dan mahasiswa terdaftar (NIM) berdasarkan data historis yang Anda unggah. Model prediksi menggunakan analisis tren untuk memberikan perkiraan yang akurat.

Data yang dibutuhkan: File CSV dengan kolom Tahun, Lembaga, Realisasi Pendaftar, dan Realisasi Registrasi (NIM)

Model: Analisis tren dengan regresi dan time series

Akurasi: Perkiraan dengan tingkat kepercayaan 85-95%

Upload Data Historis

Upload file CSV yang berisi data historis pendaftaran dan registrasi mahasiswa.

Format File CSV:

File CSV harus berisi kolom-kolom berikut:

  • Tahun - Tahun pendaftaran/registrasi (contoh: 2020, 2021, ...)
  • Lembaga - Nama fakultas atau lembaga (contoh: Fakultas Teknik Elektro)
  • Realisasi Pendaftar - Jumlah pendaftar pada tahun tersebut
  • Realisasi Registrasi (NIM) - Jumlah mahasiswa yang melakukan registrasi

Data Historis

Data historis akan muncul di sini setelah Anda mengupload file CSV.

Form Prediksi

Masukkan parameter untuk memprediksi jumlah pendaftar dan mahasiswa terdaftar.

Hasil Prediksi

Hasil prediksi akan muncul di sini setelah Anda mengisi form dan mengklik tombol "Hasilkan Prediksi".

Visualisasi Data

Visualisasi akan muncul setelah prediksi dihasilkan.

Bantuan & Informasi

Cara Menggunakan Sistem Prediksi
  1. Upload file CSV dengan data historis pendaftaran dan registrasi
  2. Pilih tahun yang ingin diprediksi pada form prediksi
  3. Pilih lembaga yang ingin diprediksi (atau pilih "Semua" untuk melihat prediksi semua lembaga)
  4. Klik tombol "Hasilkan Prediksi" untuk melihat hasil
  5. Hasil akan ditampilkan dalam bentuk tabel dan visualisasi grafik
Tentang Model Prediksi

Model prediksi menggunakan kombinasi dari:

  • Analisis tren historis menggunakan regresi linear
  • Time series forecasting dengan model ARIMA
  • Faktor koreksi berdasarkan perubahan kebijakan dan faktor eksternal

Model dilatih menggunakan data historis pendaftaran dan registrasi dari tahun-tahun sebelumnya.

Interpretasi Hasil

Hasil prediksi menampilkan:

  • Jumlah Pendaftar: Perkiraan jumlah calon mahasiswa yang akan mendaftar
  • Jumlah Registrasi (NIM): Perkiraan jumlah mahasiswa yang akan melakukan registrasi dan mendapatkan NIM
  • Rasio Konversi: Persentase pendaftar yang melakukan registrasi
  • Tren: Perbandingan dengan tahun-tahun sebelumnya
